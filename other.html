Mini Projects

Zhang Yining November 2017

This page is about the mini projects I have done in machine learning, both using basic machine learning models to learn and predict datasets. I admit that most of the work I did was writing code rather than design, but I think my code base and passion for the game can help me to be a great game designer.

Predicting Boston Housing Prices

This is a project based on available data to predict the house price of Boston, to see if the model is suitable. I used machine learning methods such as k-fold, Grid Search in this project.

I took a look at several models' learning and testing performances on various subsets of training data. The chart above produces four graphs for a decision tree model with different maximum depths.Each graph visualizes the learning curves of the model for both training and testing as the size of the training set is increased. Note that the shaded region of a learning curve denotes the uncertainty of that curve (measured as the standard deviation). The model is scored on both the training and testing sets using R2, the coefficient of determination.

Finding donors for CharityML

This is a project to find suitable fundraisers for donor agencies. Their target population is a population of more than 50,000 people a year.I employed several supervised algorithms of my choice to accurately model individuals' income using data collected from the 1994 U.S. Census. The machine learning methods I used in this project are AdaBoost, Random Forest and SVM.

From the chart above we can see that the AdaBoost is the most suitable model. The training time is much less than the other 2 methods, while the accuracy on training data is higher.I import the feature_importance_ method from SKlearn and find out the 5 most important features that contribute to the prediction. If we only use these 5 features to training the model, we can save time and get an acceptable result.

Identifying customers by clustering them

This project is based on the items purchased by customers to classify them. The machine learning methods I used in this project are K-Means and PCA.

From the chart above you can see the correlation between the purchase of certain items and other items in different dimensions so that customers can be classified. Finally, the optimal number of clusters is obtained to classify the customers and visualize the results.
