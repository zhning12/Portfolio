Plate

Teamwork Website December 2017

To help a team to connect with each other and arrange their work more suitable as well as convenient. The main part of the front-end is coded by me while the back-end and database are maintained by my teammates.

This site uses a responsive layout design. Once you change the size of the page, the page layout will be automatically adjusted to adapt to the new interface. This leads to a good presentation on the mobile and web side using the same set of code, saving a lot of code and space.

Creating a New task

You can fill in the details of the task in the corresponding form. The task page supports the users to set the deadline and upload the pictures. Users can choose the members in their team and allow them to participate in this project.

The Relationship Table and API

This is the back-end part. We devided the front-end part and the back-end part. We use the API to exchange data.

The Home Page of Plate

We can see the home page of Plate here. It introduces the general features of the site to help users quickly understand this site.

The Calendar

The calendar can show how your team arrange the work and you can clearly see your schedule.

The style of the website is Material Design by google. The main technique I use to create the website are Node.js, Vue, jQuery.
