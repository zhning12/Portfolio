GreatWatcher

Zhang Yining August 2017

The GreatWatcher is an android-based mapping trajectory APP and provide supporting introduction and data display services website. I was responsible for the Chinese and English switching function as well as the front-end part.

The System

The mobile terminal is mainly for users to use and upload the track and data, and the webpage is responsible for displaying and counting these data.

Looking at the homepage above, you can find the corresponding information in the navigation bar on the right side of the page. Mobile terminal collected data and we designed this web platform to make it easy for users to find information.

User Platform and Path Record

Users can use the mobile terminal to capture their path of travel. The figure above is an experiment we conducted while using APP records. During use, users can record and upload their travel paths on the Great Wall, so that these records can draw the shape of the Great Wall.

APP

This part was designed by our team. I would like to show you the mobile terminal. Users can draw the route on the phone, fill in the questionnaire and upload the picture.We choose a very simple line style as we think it's quite clear. The path interface is to simulate the software named 'GuDong' which is a running APP. We put the navigation bar at the bottom, which makes it easy for users to switch pages.
